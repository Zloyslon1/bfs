# Обход графа в ширину (BFS) с визуализацией

Учебный проект: алгоритм поиска в ширину (Breadth-First Search) с использованием
**очереди**. Программа пошагово показывает, как вершины добавляются в очередь и
удаляются из неё, подсвечивая текущую вершину и содержимое очереди.

## Что внутри

| Файл | Что делает |
|------|------------|
| [`bfs.py`](bfs.py) | Простая реализация BFS на Python с пошаговым выводом в консоль. Понятна новичку. |
| [`index.html`](index.html) | Красивая визуализация в браузере: граф, анимация очереди, подсветка вершин. Без установки. |

## Как запустить

### 1. Python-версия (консоль)

```bash
python bfs.py
```

Выведет каждый шаг: что извлекли из очереди, каких соседей добавили, что пропустили,
и итоговый порядок обхода.

> Используется только стандартная библиотека Python (`collections.deque`) —
> ничего устанавливать не нужно. Нужен Python 3.

### 2. Визуализация в браузере

Просто **открой файл `index.html`** двойным щелчком (или перетащи в браузер).

Возможности:
- **▶ Авто** — автоматическое проигрывание шагов;
- **Дальше / Назад** — ручное пошаговое прохождение;
- выбор стартовой вершины и регулировка скорости;
- цветовая подсветка: текущая вершина, вершины в очереди, уже обработанные,
  а также рёбра дерева обхода.

## Как работает BFS (коротко)

1. Кладём стартовую вершину в очередь и помечаем как посещённую.
2. Пока очередь не пуста:
   - извлекаем вершину из **начала** очереди (обрабатываем её);
   - всех её ещё не посещённых соседей помечаем и добавляем в **конец** очереди.
3. Очередь обеспечивает обход «по уровням» — сначала ближайшие вершины, потом дальние.

Граф в обоих файлах одинаковый, поэтому порядок обхода в консоли и в браузере совпадает.
